## 2. Life Cycle Assessment (LCA)

### ‚úÖ ALWAYS TRUE

### ‚úÖ ALWAYS TRUE

### ‚úÖ ALWAYS TRUE

### ‚úÖ ALWAYS TRUE

### ‚ùå NEVER TRUE
